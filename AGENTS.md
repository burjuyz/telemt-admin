# AGENTS.md

Этот файл не хранит полный проектный контекст.

Актуальная структура:

- обязательные правила для Cursor и AI-агентов: `.cursor/rules/*.mdc`
- описание проекта: `docs/overview.md`
- архитектурные заметки: `docs/architecture.md`
- правила локальной разработки и CI: `docs/development.md`

Если добавляете новое постоянное правило для агента, предпочитайте отдельный `.mdc` в `.cursor/rules/`, а не разрастание этого файла.
